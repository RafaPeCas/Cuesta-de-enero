Git

Git es un sistema de control de versiones distribuido que facilita el seguimiento de los cambios en el código fuente durante el desarrollo de software.

¿Para qué sirve Git?

Git se utiliza para gestionar el control de versiones de proyectos de software. Permite a los desarrolladores colaborar de manera eficiente, realizar un seguimiento de los cambios, y revertir a versiones anteriores si es necesario.

Herramientas que utilizan Git

  • GitHub
  • GitLab
  • Bitbucket
  • SourceTree
